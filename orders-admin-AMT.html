👤 관리자 이메일/코드 관리

📧 이메일 + 코드번호로 로그인합니다. 입력 시 Firebase Authentication 계정이 자동 생성됩니다.

· 총괄관리자: 전체 조회 가능, 일반관리자 이메일/코드 변경 · 추가 · 삭제 가능
· 일반관리자: 본인 코드만 조회/변경 가능 (다른 관리자는 마스킹)
· 총괄관리자 이메일은 변경 불가 (시스템 고정)

+ 새 관리자 추가 (총괄관리자 전용)
※ Firebase Auth에 자동 등록됩니다 · 최대 2명 권장

🏢 대리점 이메일/코드 관리

📧 이메일 + 6자리 코드번호로 로그인합니다. 관리자만 변경 가능합니다. 대리점 관리 탭에서 등록하면 여기에도 자동 반영됩니다.

· 총괄관리자 + 일반관리자 모두 대리점 코드 조회/변경 가능 (일반 대리점은 본인 것만 조회)

🔒 수입가격 입력 총관리자 전용

수입가격을 입력하면 부가가치세 매입세액 계산에 자동 반영됩니다.
이 정보는 총관리자(유경호)만 확인할 수 있습니다.

제품을 불러오는 중...
💡 입력 기준: 수입신고필증의 "과세가격(원)" ÷ 수입수량
• 과세가격 = CIF(제품가+운송비+보험료) + 관세
• 매입세액 = 과세가격 × 10% (수입 시 세관에 납부한 부가세)
• 이 매입세액은 매출세액에서 공제되어 실제 납부세액이 줄어듭니다
